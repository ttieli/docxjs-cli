HTML 预览功能演示 (Sample)

这是一个用于测试 DocxJS Pro HTML 预览与导出功能的样例文件。它包含了常见的网页元素及基础的 CSS 样式支持。

提示： 您可以直接在编辑器中修改此 HTML 的源码，预览区域将实时更新。

1. 文本样式

支持常见的 HTML 标签：

  • 加粗文本 (strong)
  • 斜体文本 (em)
  • 带背景色的高亮文本 (span with style)
  • 超链接测试

2. 数据表格

支持复杂的表格布局和样式定义：

原型设计 李四 进行中 65%

3. 媒体元素

支持外链图片展示：

4. 列表展示

  1. 第一步：导入 HTML 文件或粘贴源码
  2. 第二步：在预览窗口确认样式
  3. 第三步：点击“导出长图”或“导出 PDF”
